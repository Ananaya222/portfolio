#  Personal Portfolio

##  Overview

This is my personal portfolio website that showcases my education, technical skills, projects, internship experience, and contact information. It is designed to provide a simple and responsive platform to highlight my work and achievements.

---

##  Features

- Responsive design for desktop and mobile devices.
- About Me section.
- Technical Skills section.
- Projects with GitHub links.
- Internship Experience.
- Education details.
- Contact information with social links.


---

##  Technologies Used

- HTML
- CSS
- JavaScript

---

##  Project Structure

```text
Portfolio/
│── index.html
│── assets/
│   ├── images/
│   └── Resume.pdf
└── README.md
```

---

##  How to Run

1. Clone the repository.

```bash
git clone https://github.com/Ananaya222/portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
